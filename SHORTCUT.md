# Shortcut

## Common

| command | description |
| --- | --- |
| `command q` | アプリケーション終了 |

## VSCode
  
| command | description |
| :-- | :-- |
| `ctrl ;` | エディタとターミナル間の移動 |
| `command w` | 選択中のエディタのタブ削除 |
| `command b` | サイドバー表示/非表示 |

| command | description |
